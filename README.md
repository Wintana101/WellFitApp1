Well-Fit App 1 is a cross-platform mobile fitness application built with React Native and Firebase. 
The app helps users track daily steps, follow workouts, stay motivated, and monitor basic wellness habits.

## Features
- User authentication (Login & Signup)
- Personalized home dashboard
- Daily step goal tracking
- Rotating motivational quotes
- Dynamic daily workouts
- Water intake tracking
- Progress overview
- Profile management

## Technologies Used
- React Native (Expo)
- Firebase Authentication
- Firebase Firestore
- MVC Architecture
